# Building-generation-maya-plugin
Maya plugin generating parameterized buildings for a procedural city. Building models are created by combinding boxes in a controlled manner. Placement of extensions can be randomized within boundaries or calculated using a method inspierd by a shape grammar described by MÃ¼ller 2006. The calculated implementation is currently somewhat unstable.
